### 一、相关技术
- 后端：`Java`、`JavaWeb` / `Springboot`。
- 前端：`Vue`、`HTML` / `CSS` / `Javascript` 等。
- 数据库：`MySQL`

### 二、相关软件（列出的软件其一均可运行）
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- 等

### 三、功能描述
系统分为（身份）：普通用户、管理员。

**普通用户功能：**
1. 登录
2. 注册
3. 首页
4. 景点
5. 房间
6. 旅游团
7. 游玩路线
8. 公告
9. 留言板
10. 个人中心
11. 后台管理（首页、个人中心、景点管理、房间管理、旅游团管理、游玩路线管理、公告管理、留言板管理）
12. 客服


**管理员功能：**
1. 登录
2. 首页
3. 个人中心
4. 客服聊天管理
5. 基础数据管理（房间类型管理、公告类型管理、景点类型管理、旅游团类型管理、出发时间管理、出行方式管理、旅游方式管理）
6. 景点管理（景点评价管理、景点收藏管理、景点订单管理）
7. 房间管理（房间管理、房间评价管理、房间收藏管理、房间订单管理）
8. 旅游团管理（旅游团管理、旅游团留言管理、旅游团收藏管理、旅游团拼团管理）
9. 旅游游玩路线管理（游玩路线管理、游玩路线留言管理、游玩路线收藏管理）
10. 公告管理
11. 留言板管理
12. 用户管理
13. 轮播图信息
